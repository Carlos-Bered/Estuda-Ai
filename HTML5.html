O que é HTML5?

HTML5 (HyperText Markup Language versão 5) é a linguagem de marcação usada para estruturar e apresentar conteúdo na web. Ele define a forma como elementos, como textos, imagens, vídeos e links, são exibidos em um navegador. É a versão mais recente do HTML, com novos recursos e melhorias em relação às versões anteriores.


Estrutura básica de um documento HTML5

Todo documento HTML5 começa com uma estrutura padrão. Ela informa ao navegador que o arquivo segue as convenções do HTML5. Vamos dar uma olhada na estrutura básica de um documento HTML:


<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <!-- Conteúdo vai aqui -->
    <h1>Bem-vindo ao HTML5!</h1>
    <p>Este é um exemplo básico.</p>
</body>
</html>

Vou explicar cada parte

  • <!DOCTYPE html> Informa ao navegador que estamos usando HTML5.

  • <html lang="pt-BR"> Indica que o conteúdo da página está em português do Brasil.

  • <head> A seção onde você coloca metadados, como o título da página, links para arquivos CSS, e definições de charset.

  • <meta charset="UTF-8"> Define a codificação de caracteres para garantir que caracteres especiais (como acentos) sejam exibidos corretamente.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0"> Faz com que a página seja exibida corretamente em dispositivos móveis.

  • <title> Define o título que será mostrado na aba do navegador.

  • <body> Contém todo o conteúdo visível da página, como textos, imagens e links.

Principais Tags


Aqui estão algumas das tags mais usadas no HTML5:


  • Títulos: Usamos <h1> a <h6> para definir títulos e subtítulos. O <h1> é o mais importante e o <h6> o menos

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
  • Parágrafos: Para blocos de texto, usamos a tag <p>

<p>Este é um parágrafo de exemplo.</p>
  • Links: Para criar links clicáveis, usamos a tag <a>.Ela precisa do atributo href para definir o destino.

<a href="https://www.example.com">Visite o Exemplo</a>
  • Para inserir imagens, usamos a tag <img>. O atributo src define a fonte da imagem e alt é o texto alternativo (caso a imagem não carregue).

<img src="caminho/para/imagem.jpg" alt="Descrição da imagem">

Novos elementos no HTML5


O HTML5 trouxe novos elementos semânticos para melhorar a organização e acessibilidade do conteúdo. Esses novos elementos ajudam os navegadores e mecanismos de busca a entender melhor a estrutura do seu site.


Aqui estão alguns dos principais:

  • <header>: Representa o cabeçalho de uma página ou seção.

  • <nav>: Define um conjunto de links de navegação.

  • <section>: Define uma seção temática no documento.

  • <article>: Usado para artigos, posts de blogs ou conteúdos independentes.

  • <footer>: Representa o rodapé da página ou de uma seção.

<header>
    <h1>Meu Site</h1>
</header>
<nav>
    <ul>
        <li><a href="#inicio">Início</a></li>
        <li><a href="#sobre">Sobre</a></li>
        <li><a href="#contato">Contato</a></li>
    </ul>
</nav>
<section>
    <article>
        <h2>Título do Artigo</h2>
        <p>Este é o conteúdo do artigo.</p>
    </article>
</section>
<footer>
    <p>Rodapé do site</p>
</footer>

Multimídia


Uma das grandes vantagens do HTML5 é a facilidade com que ele lida com mídia, como áudio e vídeo, sem a necessidade de plugins como o Flash. Vamos ver como incluir esses elementos.


Vídeo:

Para vídeos, usamos a tag <video>


<video width="320" height="240" controls>
    <source src="video.mp4" type="video/mp4">
    Seu navegador não suporta o elemento de vídeo.
</video>

Áudio

Para áudio, usamos a tag <audio>

<audio controls>
<source src="musica.mp3" type="audio/mpeg">
    Seu navegador não suporta o elemento de áudio.
</audio>

Formulários


Os formulários são essenciais para a interação com o usuário. Eles permitem que você colete informações, como dados de login ou mensagens em um formulário de contato. Vamos ver um exemplo simples de formulário:

 
<form action="/enviar" method="POST">
  <label for="nome">Nome:</label>
  <input type="text" id="nome" name="nome" required>

        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
    
        <button type="submit">Enviar</button>
</form>

Aqui está o que cada tag faz:

  • <form>: Define o formulário. O atributo action define para onde os dados do formulário serão enviados e method="POST" define o método de envio.

  • <label>: Um rótulo para o campo de formulário.

  • <input>: Campo de entrada de dados. Os tipos mais comuns são text, email, password, entre outros.

  • <button>: Um botão para enviar o formulário.

Essa é uma visão geral básica do HTML5. Ele é a fundação de qualquer página web, e saber como estruturar corretamente seu conteúdo usando essas tags é essencial para criar sites bem organizados, acessíveis e responsivos.
