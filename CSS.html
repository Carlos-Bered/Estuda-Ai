O que é CSS?

CSS (Cascading Style Sheets) é a linguagem que usamos para definir o estilo de páginas da web. Ele permite que a gente controle a aparência dos elementos HTML, como cor, tamanho de texto, espaçamento, e a posição dos itens na página.


Estrutura básica do CSS

Vamos começar entendendo a estrutura básica de como o CSS funciona. Em CSS, nós escrevemos regras que determinam como os elementos de uma página devem ser exibidos.


A regra básica:

Cada regra CSS tem duas partes:

  1. Seletor: Esse é o nome do elemento HTML que você quer estilizar.
  2. Bloco de declaração: Dentro desse bloco, você diz o que quer mudar. O bloco tem duas partes:
    • Propriedade: O que você quer alterar (por exemplo, cor, tamanho de fonte).
    • Valor: O valor que você quer aplicar (por exemplo, red para cor, 16px para tamanho de fonte).

Aqui está um exemplo simples:



p {
  color: blue;
  font-size: 16px;
}
  • Seletor: p (isso quer dizer que estamos estilizando o parágrafo

    ).

  • Declarações: color: blue; e font-size: 16px;. Isso significa que os parágrafos terão cor azul e tamanho de fonte de 16 pixels.

Seletores CSS

Os seletores são usados para "alvo" dos elementos HTML que você quer estilizar. Alguns exemplos de seletores:

  • Seletor de elemento: Estiliza todos os elementos daquele tipo.

h1 {
  color: green;
}
  • Seletor de classe: Aplica um estilo a elementos que têm uma determinada classe. As classes começam com um ponto (.).

.meu-texto {
    font-weight: bold;
}
  • Seletor de ID: Aplica o estilo a um único elemento com um ID específico. IDs começam com o símbolo de cerquilha (#).

#meu-titulo {
    text-align: center;
}

Propriedades mais comuns

Aqui estão algumas das propriedades mais usadas no CSS:

  • Cor do texto: color

p {
  color: red;
}
  • Tamanho da fonte: font-size

p {
  font-size: 20px;
}
  • Margens: margin (define o espaço externo ao redor de um elemento)

div {
  margin: 20px;
}
  • Padding: padding (define o espaço interno de um elemento, entre o conteúdo e a borda)

div {
  padding: 10px;
}
  • Borda: border (define as bordas ao redor de um elemento)

div {
  border: 2px solid black;
}

Como o CSS é aplicado?

Você pode aplicar o CSS de três maneiras diferentes:

  • CSS Inline: Dentro de uma tag HTML, usando o atributo style:

<p style="color: blue;">Este é um parágrafo azul</p>
  • CSS Interno: Dentro de uma tag <style> no cabeçalho da página HTML:

<style>
p {
  color: blue;
}
</style>
  • CSS Externo: Arquivo CSS separado que é referenciado na página HTML. Esse é o método mais recomendado, porque mantém o código organizado.

<link rel="stylesheet" href="styles.css">

Cascata e Especificidade

O "C" em CSS significa "cascading" (cascata), o que quer dizer que as regras podem se sobrepor. Se houverem múltiplas regras aplicadas a um mesmo elemento, o navegador vai decidir qual usar baseado na especificidade. Quanto mais específico for o seletor, maior a prioridade ele terá.


Por exemplo:


p {
  color: red;
}
#meu-paragrafo {
    color: blue;
}

Se um parágrafo tiver o ID meu-paragrafo, ele será azul, porque a regra com o ID é mais específica do que a regra que estiliza todos os <p>.



lembre-se de que o CSS é essencial para a apresentação de páginas web.Em resumo, dominar o CSS é fundamental para criar designs atraentes e funcionais.
